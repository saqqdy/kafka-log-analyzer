# kafka-log-analyzer 发版计划

## 一、版本规划

### 版本命名规范

```
主版本.次版本.修订版本 (Major.Minor.Patch)

- Major: 架构重大变更、不兼容升级
- Minor: 新增功能、向后兼容
- Patch: Bug 修复、小改进
```

### 版本路线图

| 版本 | 里程碑 | 核心目标 | 预计时间 |
|------|--------|----------|----------|
| **v0.1.0** | Alpha | Plugin 骨架 + 基础分析能力 | 第 1 周 |
| **v0.2.0** | Alpha | MCP Server + 命令完善 | 第 2 周 |
| **v0.3.0** | Beta | Prometheus 数据源集成 | 第 3-4 周 |
| **v0.4.0** | Beta | Hook 自动触发 + 推送集成 | 第 5-6 周 |
| **v0.5.0** | RC | 历史对比 + 趋势分析 | 第 7 周 |

| **v1.1.0** | 增强 | 用户体验优化 | 第 9-10 周 |
| **v2.0.0** | 下一代 | AI 根因预测 + 自动修复 | Q2 |

## 三、发版流程

### 分支策略

```
main (稳定分支)
  │
  ├── develop (开发分支)
  │     │
  │     ├── feature/xxx (功能分支)
  │     ├── fix/xxx (修复分支)
  │     └── release/vX.Y.Z (发版分支)
  │
  └── hotfix/xxx (紧急修复分支)
```

### 发版检查清单

#### 发版前（Pre-release）

- [ ] 所有测试通过（`npm test`）
- [ ] TypeScript 编译无错误（`npm run build`）
- [ ] Lint 检查通过（`npm run lint`）
- [ ] 更新 CHANGELOG.md
- [ ] 更新版本号（package.json、plugin.json）
- [ ] 创建 Git tag（`vX.Y.Z`）
- [ ] 代码审查通过（至少 1 人）
- [ ] 文档更新完成

#### 发版中（Release）

- [ ] 构建生产产物
- [ ] 打包 Plugin（`npm run pack`）
- [ ] 上传到发布渠道
- [ ] 创建 GitHub Release
- [ ] 发布 Release Notes

#### 发版后（Post-release）

- [ ] 监控错误率
- [ ] 收集用户反馈
- [ ] 更新文档网站（如有）
- [ ] 关闭相关 Issue

## 七、回滚策略

### 回滚场景

| 场景 | 触发条件 | 回滚动作 |
|------|----------|----------|
| **严重 Bug** | P0 Bug 影响 | 回滚到上一版本 |
| **性能下降** | 性能低于基线 50% | 回滚到上一版本 |
| **配置不兼容** | 升级后无法启动 | 回滚配置 + 版本 |
| **数据损坏** | 持久化数据异常 | 回滚版本 + 恢复备份 |

### 回滚流程

```bash
# 1. 停止当前版本
pm2 stop kafka-analyzer

# 2. 回滚到上一版本
npm install kafka-log-analyzer@0.4.0

# 3. 回滚配置（如有必要）
git checkout v0.4.0 -- config/

# 4. 启动旧版本
pm2 start kafka-analyzer

# 5. 验证回滚成功
curl http://localhost:3000/health
```

## 十一、风险预案

### 发布风险

| 风险 | 影响 | 概率 | 应对措施 |
|------|------|------|----------|
| 发布失败 | 用户无法安装 | 低 | 提前测试安装流程 |
| 版本回退需求 | 用户投诉 | 中 | 保持旧版本可用，提供回滚指南 |
| 配置不兼容 | 升级失败 | 中 | 提供配置迁移脚本 |
| 依赖冲突 | 安装失败 | 低 | 锁定依赖版本，提供 peer deps |
| 性能下降 | 用户体验差 | 中 | 性能基准测试，回滚策略 |

### 回滚触发条件

```
发布后 24h 内：
- 错误率 > 5%（触发回滚）
- P0/P1 Bug 超过 3 个（触发回滚）
- 用户投诉超过 5 个（评估回滚）

发布后 7 天内：
- P0 Bug 未修复（触发 hotfix）
- 性能下降 > 30%（评估回滚）
```
